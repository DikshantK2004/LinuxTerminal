In order to run the code , run the makefile using make command
then type ./myShell and hit Enter

System is not implemented in this terminal!!!!

Commands Implemented:
1.) list - lists all files of a folder , options -[l S t a r]  
2.) copy - copies files , options -[i n u v]
3.) move - moves files , options -[i n u v]
4.) process - lists processes , options -[a f F e]
5.) search - searches for patterns , options -[w i n c]
6.) exit - exits from the terminal
7.) chngdir - changes directories


-> Type help list for more info on list
-> Type help copy for more info on copy
-> Type help move for more info on move
-> Type help process for more info on process
-> Type help search for more info on search
-> Type help exit for more info on exit
-> Type help chngdir for more info on chngdir
